# &-Inch Quad MAVLik Mission

## Setup
1. Clone the repo:
    ```
    git clone https://github.com/uci-uav-forge/uavf_2025.git
    cd 7_Inch_MAVLink_Mission
    ```

2. Now that you are in the project, create the virtual environment:
   ```
    python3 -m venv venv
    source venv/bin/activate
    pip install -e .
   ```
